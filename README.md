# awan

A minimalist single-page typography site with animated text reveal.

## Tech Stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com) (via CDN)
- Google Fonts: Cormorant Garamond, DM Sans
- Vanilla JavaScript animations

## Features

- Staggered character-by-character text reveal animation
- Elegant serif display font with sans-serif body text
- Subtle ambient radial glow background
- Responsive typography using `clamp()`
- No build step required — open `index.html` directly in browser

## Quick Start

```bash
# Open directly in browser
open index.html
```
